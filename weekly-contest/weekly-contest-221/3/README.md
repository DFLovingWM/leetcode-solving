# 5210. 球会落何处

## 并查集

思路类似于题目[959. 由斜杠划分区域](https://leetcode-cn.com/problems/regions-cut-by-slashes/)：

将每个格子4等分（top/right/bottom/left）。对于每个格子，建立自身、与相邻格子的连通性，最终判断「首行top」与「尾行bottom」的连通性。

但由于并查集建立的是纯粹的「连通性」；而本题的球是有「走向」的，即永远都「无法往上走」，所以不符合。
