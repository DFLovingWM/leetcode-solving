# 抛掷硬币

## 思路

典型的动态规划题目。

做题时可能想到Top-down，但实际Bottom-up更好理解：

`dp[i][j]`表示前i个骰子中，有j个向上的概率。则状态转移方程为：

```js
if (j === 0) {
  dp[i][0] = dp[i-1][0] * (1 - probs[i-1]) // 前(i-1)个向下，这个也向下
} else {
  dp[i][j] = dp[i-1][j-1] * probs[i-1] + // 这个向上
    dp[i-1][j] * (1 - probs[i-1]) // 这个向下
}
```

当然，Bottom-up还能对压缩到一维。
