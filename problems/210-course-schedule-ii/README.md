# 210. 课程表II

## 题意

题意与限制不变，这次需要输出课程的学习顺序。

## 思路

思路参考版本1：求拓扑排序。如果序列长度小于N表示有环，返回空数组即可。

## 反思总结

拓扑排序的用途：

- 检测环：
  - BFS：若最终存在结点无法删除，则有环。
  - DFS：如果又遍历到visiting状态的结点，说明有环。（有向）
- 求环路径：
  - BFS：最终无法删除的结点即为环。（无向）
  - DFS：用path记住当前遍历路径。（有向）
- 求DAG的拓扑序列（有向、无环）
  - BFS：删除的顺序构成拓扑序列。
  - DFS：结点在后序操作（回溯）时加入路径，最终路径的逆序构成拓扑序列。
