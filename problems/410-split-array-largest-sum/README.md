# 410. 分割数组的最大值

## 思路

最小化最大值问题。二分枚举最大值，然后看份数是否在K之内。这道题的贪心意义有点难理解，可以参考几乎完全相同的另一道题：[1011. 在 D 天内送达包裹的能力](https://leetcode-cn.com/problems/capacity-to-ship-packages-within-d-days/)。
