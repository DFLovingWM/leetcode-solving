# 343. 整数拆分

## 思路

### 动态规划

`dp[n]`表示和为`n`对应的最大乘积。

### 贪心

为了使乘积最大，尽可能拆分为多个因子。不过还是先观察：

```
f[1] = 1
f[2] = 1 <== 1 * 1
f[3] = 2 <== 2 * 1
f[4] = 4 <== 2 * 2
f[5] = 6 <== 2 * 3
f[6] = 9 <== 3 * 3
f[7] = 12 <== 3 * 2 * 2
f[8] = 18 <== 3 * 3 * 2
```

可以得出结论：

- 在`n = 4`之前有`f[n] > n`，说明因子到了`2`或`3`，就可以停止了/不必再继续拆分了。反之，在`n = 4`及之后，需要继续拆分。
- 尽可能拆分为`2`与`3`这两种因子。
