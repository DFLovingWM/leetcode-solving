# 841. 钥匙和房间

## 题意

给定N个房间，以及房间0的钥匙。每进入一个房间，可能获取新的钥匙。问是否所有房间都能打开。

## 思路

房间即结点，钥匙即边。题目转化为：从0开始，求所有房间是否都可达。方法：

- 搜索/着色（DFS/BFS）
- 并查集（好像没必要）
- 模拟

使用DFS是最合适的，因为进入房间相当于遍历一个点，而获取新钥匙相当于扩展子结点。
