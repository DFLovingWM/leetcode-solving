# 230. 二叉搜索树中第K小的元素

## 题意

给定一个二叉搜索树，求第K小的元素。

## 思路

- 中序遍历：遍历的同时计数，计数到K则返回当前结点。
  - 递归：O(K)
  - 迭代：O(H + K)
- 二分搜索：当前结点（相当于二分查找中的middle）是第C小（C = 左子树的结点数量 + 1）。如果C > K，往左走；如果C < K，往右走，但注意此时要找第(K - C)小。
  - 时间：O(NH)
