# 279. 完全平方数

## 题意

给定一个数字n，找出和为n的完全平方数的最小组合数。

## 思路

和只能由完全平方数组成。如果数字A、B之间相差了若干个完全平方数，则B可以由A组成。这种推导关系，实际上就是动态规划的推导式。

### Bottom-up DP

举个例子，考虑当n=12时，有以下几种可能：

- 相差1：dp[12] = dp[11] + 1
- 相差4：dp[12] = dp[8] + 4
- 相差9：dp[12] = dp[3] + 9
- 相差16：不可能，搜索停止

然后为了求最小组合个数，取个min即可。

总结一下：`dp[i]`表示和为`i`的最小组合数，有状态转移方程：

```js
dp[i] = min(dp[i - d]) // d为小于等于i的所有完全平方数
```

### Top-down DP

思路同Bottom-up DP，函数设计与dp数组一致。终止条件也一样：f(0)返回0。

### BFS

求最小组合个数，其实也是在求最短路径，所以可以用BFS来解。最好按照【层次遍历】来编码，这样遇到n时的层次数，就是答案。

### 数学法

以上答案都是O(N^2)的解法，还有一种更快速的解法：拉格朗日四平方数和。
